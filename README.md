# Django Quick Start
    1. 가상화 설치
    python3 -m venv venv

    2. django 설치
    python -m pip install Django

    3. django 설치 확인
    python3
    >> import django

    4. django 버전 확인
    python3 -m django --version

    5. 가상화 실행
    source venv/bin/activate
    
    6. 파일 구동
    cd devmap
    python3 manage.py runserver
